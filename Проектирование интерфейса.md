| Элемент |	Назначение |
| :------------- | :------------- |
| DataGrid или ListView |	Таблица со списком контактов |
| Панель ввода/редактирования	| Поля: ФИО, Телефон, Email, Категория (ComboBox) |
| Кнопки | Добавить, Редактировать, Удалить, Очистить поля |
| Поле поиска |	TextBox + кнопка «Найти» |
| Фильтр по категориям | ComboBox или список радио-кнопок  |
| Строка состояния | Количество контактов, статус операции |
| Меню | Файл (Сохранить, Экспорт, Выход), Справка |
